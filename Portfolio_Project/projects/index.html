Data Visualization Project at Grainger

For this project, I used Power BI to create a dynamic dashboard analyzing property tax data. I developed an automated Power BI dashboard integrated with Excel, providing efficiency and value for the finance team.

The app also includes:

  • Dynamic visuals for decision-making

SkiShare Mobile App

I created SkiShare, a mobile app designed for ski enthusiasts, similar to Slopes, but with unique features. One standout feature is the ability to share photos with others skiing on the same mountain on the same day, creating community engagement among skiers.

The app also includes:

  • Real-time trail tracking
  • Performance analytics
  • Interactive community features
